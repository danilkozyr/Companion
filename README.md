# 42Plan
iOS application based on 42 API to search users in school system and display user's information.

#### Features
- Search user
- Get detailed info about user

#### API Used:
- [42API](https://github.com/danilkozyr/42Plan)

#### Frameworks Used:
- Alamofire
- SwiftyJSON
